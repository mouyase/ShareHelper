# 分享助手

一个极简 Android 分享中转工具。它不提供主界面，会出现在系统分享面板和图片/视频的「打开方式」里：接收图片或视频，生成一份临时处理副本，再立刻调起系统分享面板交给其他 App。

## 它会做什么

- 支持单个或批量分享，图片/视频混合也可以。
- 支持从系统分享入口接收图片/视频，也支持作为图片/视频资源的打开目标。
- 每个输入都会导出为一个独立 HTML 文件；处理后的文件只放在 app cache，不保存到相册或公共目录。
- 图片 HTML 内会直接嵌入 base64 PNG data URL：图片会按 EXIF 方向校正后嵌入，不再单独分享 PNG 文件。
- 视频 HTML 内含 `<video controls>` 播放器和嵌入的 base64 视频数据；MP4 会优先重新封装并写入 metadata track 后再嵌入，其他视频或重新封装失败时会嵌入保守临时副本。
- 处理中会显示一个轻量 loading 弹窗，弹窗配色会跟随系统亮色/暗色模式。
- 为避免缓存被撑爆，HTML 视频导出会按当前可用缓存空间动态判断；只要空间足够就尽量处理。
- HTML 图片和 HTML 视频在不同接收 App 中的预览和播放兼容性取决于对方实现，属于尽力而为。

## 怎么用

安装后，在相册或文件管理器里选择图片/视频，可以点系统分享并选择「分享助手」，也可以在「打开方式」里选择「分享助手」。处理完成后，它会再次打开系统分享面板。

## 构建

```bash
./gradlew :app:assembleDebug
```

## 发布

推送 `v*` tag 会触发 GitHub Actions 构建 signed release APK。发布产物命名为 `ShareHelper-v<version>.apk`。

本地发布脚本：

```bash
scripts/publish-release.sh 0.0.1
```
